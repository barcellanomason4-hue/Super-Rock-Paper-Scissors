# Super-Rock-Paper-Scissors
This is a modified version of rock, paper, scissors that has special screens for when you lose or win
Difficulty is straight forward!!!! It gets harder per round, so think about your choices before picking!!!!
SPECIAL SCREENS HAVE SPECIFIC COLORS FOR THE RESULTS (they have a pulsing effect)
RED MEANS YOU WIN
BLUE MEANS TIE
GREEN MEANS YOU LOSE
